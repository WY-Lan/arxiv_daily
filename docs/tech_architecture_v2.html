🔄 Execution Pipeline
1. Fetch
PaperFetcherAgent.execute()
→ arxiv_api.fetch_daily_papers()
→ semantic_scholar.get_papers_batch()
→ pwc.search_paper_by_arxiv()
→ db.save_paper()
2. Select
SelectionAgent.execute()
→ _evaluate_papers()
  ↳ citation_score
  ↳ influence_score
  ↳ quality_score (LLM)
  ↳ community_score
→ _select_top_papers(n=10)
3. Summarize
SummaryAgent.execute()
→ llm.generate_json()
  ↳ core_contributions
  ↳ key_innovations
  ↳ recommendation_reason
→ context.set("summaries")
4. Publish
PublisherAgents.execute()
→ Notion (MCP)
→ Feishu (Webhook)
→ XHS (MCP)
→ WeChat (API)
Collection mode aggregation
