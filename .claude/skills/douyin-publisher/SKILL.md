# 抖音图文发布技能

将AI Agent论文转化为适合抖音平台的图文内容，采用抖音特色风格：竖版封面、大字号标题、醒目推荐标签。

## 内容格式

**输出格式**：
```json
{
  "title": "标题（20字以内，吸引眼球）",
  "content": "正文（分段清晰，信息密度高，300-600字）",
  "tags": ["#AI", "#Agent", ...]
}
```

## 标题规则

抖音图文标题需要吸引眼球，常用技巧：
- 数字开头：「5篇Agent论文，看完涨知识」
- 疑问句：「这些Agent论文凭什么火？」
- 感叹句：「最新Agent论文，太强了！」
- 对比句：「同样做Agent，差距在哪？」
- 单位+产品：「北大推出「记忆Agent」！」

**限制**：标题不超过20字

## 正文风格

抖音用户习惯快速浏览，正文需要：
- 段落短小（每段50-100字）
- 重点突出（使用💡、🔥、⚡️等emoji）
- 信息密度高但易读
- 结尾引导关注

**示例结构**：
```
今日精选X篇Agent论文，干货满满！

📄 论文1：标题
核心创新：一句话概括
关键突破：技术亮点

📄 论文2：标题
...

关注我，每日推送AI前沿论文！
```

## 标签策略

抖音标签推荐：
- 热门话题：#AI #人工智能 #科技
- 领域标签：#Agent #大模型 #ChatGPT
- 内容标签：#论文 #科研 #学习
- 吸引标签：#知识分享 #干货

**数量**：3-5个标签

## 封面风格

抖音图文使用竖版封面（1080x1920）：
- 大字号标题（60-80px）
- 醒目的「今日推荐」标签
- 论文封面网格拼接
- 鲜艳的背景色

## 技术集成

### MCP工具
使用 `douyin-upload-mcp-skill` 项目提供的工具：
- `douyin_publish_imagetext` - 发布图文
- `douyin_check_login` - 检查/推进登录流程
- `douyin_screenshot` - 调试截图

### 图片要求
- 支持1-35张图片（抖音平台限制）
- 推荐使用竖版封面（1080x1920）
- 图片格式：PNG/JPG
- 图片命名格式：`{arxiv_id}_cover.png`

## 发布流程

1. 生成抖音风格内容（使用本技能的提示词）
2. 生成竖版封面图（使用 tools/douyin_cover.py）
3. 通过 MCP 发布图文
