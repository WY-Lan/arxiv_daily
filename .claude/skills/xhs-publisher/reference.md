# 小红书发布参考文档

## 平台特性

### 用户画像
- 年龄：18-35岁为主
- 性别：女性用户占比约70%
- 兴趣：生活方式、学习成长、职场技能
- 阅读习惯：碎片化、图文结合

### 内容偏好
- 标题党适度：吸引眼球但不夸张
- 干货导向：实用价值高的内容受欢迎
- 视觉美观：排版整洁、配图精美
- 互动性强：引导评论、点赞、收藏

## 正文写作规范

### 开头技巧
```
✅ 直接点题：
"研究Agent的看过来！"

✅ 痛点引入：
"还在为Agent开发头疼？"

✅ 数据吸引：
"这篇论文引用5000+！"

❌ 平淡无奇：
"今天介绍一篇论文..."
```

### 内容结构
```
1. 开头（50字）：吸引注意力
2. 核心内容（200-400字）：价值输出
3. 应用价值（100字）：实际意义
4. 结尾（50字）：引导互动
```

### 段落规范
- 每段不超过100字
- 使用emoji分隔（每段最多1个）
- 重点内容加粗
- 使用列表增强可读性

## 违规词规避

### 禁用词汇
- 绝对化用语：最好、第一、唯一
- 虚假宣传：包治百病、立竿见影
- 引流词：加微信、私信领、扫码

### 替代方案
```
❌ 最好的Agent框架

```

## 常见问题

### Q: 发布后为什么没有流量？
A: 检查标题吸引力、封面图质量、发布时间

### Q: 如何提高互动率？
A: 在结尾设置问题、引导评论、及时回复

### Q: 标签如何选择？
A: 结合热门标签和垂直领域标签，6-8个为宜
