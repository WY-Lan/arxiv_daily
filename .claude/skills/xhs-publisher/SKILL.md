# 小红书发布技能

将AI Agent论文转化为适合小红书平台的笔记内容，支持单篇深度解读和合集汇总两种模式。

## 发布模式

### 模式一：单篇发布 (single)

适用于重点论文的深度解读，学术化风格。

**输出格式**：
```json
{
  "title": "标题（单位推出「核心概念」产品名 + emoji）",
  "content": "正文内容（段落式，空行分隔，400-800字）",
  "tags": ["#AI论文", "#Agent", ...]
}
```

**标题规则**：
- **必须包含单位信息**（从摘要提取）
- 格式：单位 + "推出" + 「核心概念」产品名 + emoji
- 多个单位用 "&" 连接
- 示例："港中文&华为推出「记忆增强」扩散语言模型🔥"

**正文风格**：
- 段落式，空行分隔，每段一个要点
- 学术化、紧凑、有深度
- 适当使用 emoji 突出重点（💡、🎯、⚡️等）
- 结尾格式：paper：论文标题
- **不需要互动引导**

### 模式二：合集发布 (collection)

适用于每日多篇论文的汇总推送。

**输出格式**：
```json
{
  "title": "标题（20字以内）",
  "content": "正文（500-800字，包含多篇论文简介）",
  "tags": ["#AI论文", "#Agent", ...]
}
```

**内容结构**：
1. 开场白：概括今日主题（50-100字）
2. 论文列表：每篇80-150字简介
3. 结尾互动：引导互动（30-50字）

## 写作风格指南

### 标题技巧
- 数字吸引：「今日5篇Agent论文精选」
- 疑问句：「这些Agent论文凭什么火？」
- 对比句：「同样做Agent，差距在哪？」
- 热点结合：「2024 Agent论文必读合集」

### 正文要求
- 专业术语通俗化
- 突出实际应用价值
- 适当使用emoji（标题1-2个，正文每段最多1个）
- 避免过度标题党，内容要有干货

### 标签策略
- 平台热门：#AI论文 #人工智能 #机器学习
- 领域标签：#Agent #大模型 #LLM
- 内容标签：#论文分享 #学术 #科研
- 吸引标签：#干货分享 #学习打卡

## 技术集成

### MCP工具
- `mcp__xiaohongshu-mcp__publish_content` - 发布图文笔记
- `mcp__xiaohongshu-mcp__check_login_status` - 检查登录状态

### 图片要求
- 支持1-18张图片（小红书平台限制）
- **推荐**：提取PDF所有页面作为图片（最多18页）
- 图片路径支持本地绝对路径或HTTP/HTTPS链接
- 图片命名格式：`{arxiv_id}_page_{n}.png`
